JavaScript has the [comma operator (`,`)](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators/Comma_operator) that evaluates its operands from left to right and returns the value of the last operand, discarding results of other operands.

```js
let a, b, c;
console.log((a = b = 3, c = 4)); // logs 4
// a = 3
// b = 3
// c = 4
```
