GitHub has a feature that allows you to turn any user or repo activity into an Atom news feed.

URL format for different types of feeds:

- Repo releases: `https://github.com/:owner/:repo/releases.atom`
- Repo commits: `https://github.com/:owner/:repo/commits.atom`
- Repo tags: `https://github.com/:user/:repo/tags.atom`
- User activity: `https://github.com/:user.atom`
- Private feed: `https://github.com/:user.private.atom?token=:secret`
